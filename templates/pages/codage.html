Why learn to code?

Computers are part of our everyday life, we use them to surf the internet, do a Google search, see pictures, read e-mails, play games or write text. But how they do all of this? Is it magic? Yes, it's magic. A magic called coding (or programming). Learning to code is not about vocation but a skill necessary to understand how our world works.

Now days learn to code is not about vocation but a required skill in order to understand how our world works.

Positive aspects

Programming is an activity that stimulates creativity, promotes collaboration, develops rigor, patience and perseverance. It trains a logical thinking to solve problems.

Methodology

In a fun and enjoyable way you will become familiar with the basic concepts of coding: variables, loops, conditions, etc. We will make you discover different interfaces based on visual blocks where you can create animations, mini-games or your own stories.
